What is BoomStick™ API Framework?

The BoomStick™ API Framework is a Plugin for Grails on Spring Boot Microservice. It is a Reactive API Framework allowing improved automation, code reduction, performance and scalability for distributed architectures and was built by a leader in the API industry who has spent years solving issues with API installations in distributed architectures.

What Are It's Benefits?

The BoomStick™ API Framework is based upon the principles of API abstraction and shared IO state. These create an entirely NEW API Pattern allowing us the ability to do the following:

  • 60-80% code reduction for API development
  • Vastly simplified API development
  • On-the-fly reloading of endpoint data
  • Ability to sychronize endpoint data across all services
  • Automated checks
  • Automated batching
  • More scalable
  • Reduction in dropped threads
  • 100% stateless

Creating pages manually

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
