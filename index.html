What is BoomStick™ API Framework?

The BoomStick™ API Framework is a Plugin for Grails on the Spring Boot Microservice.

It installs a fully Reactive API Framework allowing improved automation, code reduction, simplified/fast development, and improved performance/scalability for distributed architectures.

It was built by a leader in the API industry who has spent years solving issues with API installations in distributed architectures and who has advised/consulted/worked with enterprise leaders such as Apple, Netflix, VMWare, Cisco, Mashery, Amazon and others.

What Are It's Benefits?

The BoomStick™ API Framework is based upon the principles of API abstraction and shared IO state. These create an entirely NEW API Pattern allowing us the ability to do the following:

  • 60-80% code reduction for API development
  • Vastly simplified API development
  • On-the-fly reloading of endpoint data
  • Ability to sychronize endpoint data across all services
  • Automated checks
  • Automated batching
  • API Chaining
  • More scalable
  • Reduction in dropped threads
  • 100% stateless

Ease of API Development

All communication logic and data are abstracted away from the controllers and services leaving the controllers to only deal with retrieving and returning a resource. This makes API development INCREDIBLY simple. You simply have to:

  • Install the plugin
  • Build your controllers and domains
  • Build a JSON IO State file for each controller
  • Compile project

We are even working on automating the rest of this as well. :)
